## Technical debt discovered

1. **Live-market flakiness for open→close demos** — the dummy evaluator
   depends on the latest candle. A timeframe that is SELL during probing can
   flip to BUY before evaluation. Prefer higher timeframes (e.g. 4h) for
   deterministic browser demos, or introduce a test evaluator later.
2. **Conceptual `PaperPosition` naming overlap with RC-16** — no TypeScript
   conflict; documented in 030. Do **not** rename for this baseline.
3. **In-memory paper state** — positions/history reset on process restart by
   design.
4. **Historical story-number collision** — roadmap still lists an older
   “US010 — First Donchian Campaign”; Architecture `030-US010` is the Signal/
   Paper foundation story. Keep the Architecture numbering as source of truth
   for this baseline.
5. **Pre-existing OutboxPollingService noise** — unrelated Prisma “record not
   found” errors may appear in API logs during long-running `nest start --watch`.
